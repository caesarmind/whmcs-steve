What is SiteLock?
SiteLock is a complete website security platform — daily vulnerability scans, automatic malware removal, Web Application Firewall, and a Trust Seal visitors can verify. Everything you need to stop cyber attacks and keep your site clean, all in one product.
Do I need technical skills to use it?
None. Setup takes under 5 minutes. Point your domain, verify ownership, and SiteLock handles scanning, alerting, and remediation automatically.
What is a virus scanner?
A virus scanner inspects every file on your website against a known malware database and behavioral heuristics. Any infected files are quarantined and the site admin is notified immediately.
How often does SiteLock scan?
Find plans scan daily. Fix plans scan 3× daily. Defend runs continuous monitoring with real-time alerting on any file change.
What’s an SQL Injection malware pattern?
SQL injection attacks inject malicious SQL statements into input fields, tricking the database into returning or destroying data. SiteLock’s WAF blocks these patterns in real time at the edge.
What about XSS and DDoS?
XSS (cross-site scripting) and DDoS attacks are handled by the Defend-tier WAF and edge mitigation — absorbing up to 1 Tbps before traffic reaches your origin.
What is the SiteLock Trust Seal?
The Trust Seal is a clickable badge you display on your site. It shows live scan status and is recognized globally by search engines and customers — lifting conversions.
Can I customize the Trust Seal?
Fix and Defend tiers include customization: size, color variant, and placement. Light and dark variants are included.
