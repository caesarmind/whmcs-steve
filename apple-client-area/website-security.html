Security questions.

Common questions about website protection.

How often are scans performed?
All plans include daily malware scans. Premium plans add real-time monitoring that checks your site continuously throughout the day for any changes or threats.
What happens if malware is found?
Professional and Premium plans include automatic malware removal. Basic plans alert you immediately with a detailed report showing exactly which files are affected.
Will it slow down my website?
No. Our security tools run server-side and have zero impact on page load times. The CDN feature actually makes your site faster by caching content at edge locations.
Do I need technical knowledge?
Not at all. Setup takes under 5 minutes. Simply add your domain, verify ownership, and our systems handle everything else — scanning, monitoring, and remediation.
