Do I need any coding skills to use Site Builder?
No coding skills are required. Site Builder uses a visual drag-and-drop editor that lets you design your website in real time. Simply choose a template, customize it with your content, and publish. Advanced users can add custom code if needed.
Can I use my own domain name?
Yes, you can connect any custom domain to your Site Builder website. If you already have a domain registered with us, connecting it is a one-click process. External domains can be connected by updating your DNS settings.
What is the white-label option?
The white-label option, available on the Agency plan, removes all Site Builder branding and lets you present the builder under your own brand. This is ideal for agencies and freelancers who build websites for clients and want a seamless branded experience.
Is my website mobile-responsive?
Yes, all websites built with Site Builder are fully responsive out of the box. The editor includes a mobile preview mode so you can see exactly how your site looks on phones and tablets before publishing. You can also customize the mobile layout independently.
