Frequently Asked Questions

How does the malware scanner work?
Our malware scanner performs automated daily scans of your website files, database, and server environment. It compares files against known malware signatures and uses heuristic analysis to detect suspicious behavior. When threats are found, you receive an immediate alert with details and remediation steps.
Will website security slow down my site?
No. Our Professional and Enterprise plans include a CDN (Content Delivery Network) that actually speeds up your website by caching content at edge locations worldwide. The firewall and security scanning run in the background without impacting page load times.
What happens if my site gets hacked?
Professional and Enterprise plans include automatic malware removal. Our system will detect and clean infected files, then notify you of what was found and fixed. Enterprise customers also have access to a dedicated security analyst for manual investigation and recovery assistance.
Do I still need an SSL certificate with Website Security?
Yes. Website Security and SSL certificates serve different purposes. SSL encrypts data in transit between your server and visitors, while Website Security protects your server from attacks, malware, and vulnerabilities. We recommend using both for comprehensive protection.
